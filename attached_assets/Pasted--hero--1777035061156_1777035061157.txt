Нужно реализовать на hero-секции сайта динамический визуальный эффект как на референсе.

Цель:
Сделать тёмно-синий hero background с ярким голубым источником света за центральной четырёхконечной звездой. Звезда должна быть тёмнее/синее на фоне свечения, а за ней должны появляться мягкие лучи света/тени, которые динамически меняются при движении источника света.

Визуал:
- Фон: глубокий тёмно-синий / navy gradient.
- В центре — большая четырёхконечная звезда, похожая на compass/star shape.
- За звездой — яркое cyan/blue свечение.
- От звезды должны расходиться мягкие лучи/тени, будто объект перекрывает источник света.
- Лучи не должны быть резкими. Нужен мягкий cinematic blur, smooth glow.
- Эффект должен выглядеть premium, minimal, fintech/crypto/institutional style.
- Цвета: dark navy, royal blue, cyan, soft electric blue.
- Эффект должен быть адаптивным и хорошо выглядеть на desktop и mobile.

Динамика:
- Источник света должен медленно двигаться за звездой.
- Движение может быть автоматическим по плавной траектории: left/right/up/down или circular/elliptic.
- Лучи/тени должны визуально следовать за движением источника.
- Анимация должна быть плавной, без дёрганий.
- Скорость медленная: примерно 8–14 секунд на цикл.
- Эффект должен работать постоянно в фоне hero-секции.

Технические требования:
- Реализовать на React + Tailwind.
- Предпочтительно CSS/SVG/HTML без тяжёлых библиотек.
- Можно использовать CSS variables для позиции источника света:
  --light-x
  --light-y
- Позиция источника света должна управлять:
  1. radial-gradient свечением;
  2. направлением/положением лучей;
  3. тенями вокруг звезды.
- Использовать absolute layers:
  - base dark gradient background;
  - animated light glow layer;
  - rays/shadow layer;
  - central star shape;
  - optional grain/noise overlay;
  - hero text above/around effect.
- Звезду можно сделать через SVG или clip-path.
- Лучи можно сделать через pseudo-elements, conic-gradient, radial-gradient, mask-image или SVG filters.
- Добавить blur, opacity и mix-blend-mode, чтобы лучи выглядели мягко.
- Не перегружать GPU: использовать transform, opacity, filter умеренно.
- Добавить pointer-events-none на декоративные слои.
- Учитывать prefers-reduced-motion: если у пользователя отключены анимации, оставить статичный красивый свет.
